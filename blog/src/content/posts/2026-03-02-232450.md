最近、Claude Code から離れて Codex を使い始めたので、勉強がてら作ってみました。

トレンドを集める犬ということで、名前は `trendog` 🐶 です。  
まだ動作確認しきれていないので、現時点では Beta 版という位置づけです。

https://github.com/HiroVodka/trendog

## 何をするBotか

`trendog` は、以下の流れでトレンド情報をSlackに通知するBotです。

1. Hacker News / はてなブックマーク / Zenn から記事を収集  
2. Gemini で重要度を判定  
3. 重要度が高い記事だけ要約  
4. Slack に投稿

## 使ってみた感触

今のところ API Key は Gemini の無料枠で足りそうですが、運用を続けるならコストは継続的に見ていきたいです。

また、この手のBotはすでにいろいろな実装例があるので、差別化として Alexa スキル化を検討しています。  
毎朝 Alexa 経由で要約を聞ける形にできると、使い勝手が上がりそうです。
